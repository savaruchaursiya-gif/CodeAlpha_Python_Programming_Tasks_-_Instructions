# 🎮 Task 1 – Hangman Game

## 📝 Description

The **Hangman Game** is a GUI-based word guessing application developed using **Python and Tkinter**. The game randomly selects a hidden word from a predefined list, and the player must guess the word by selecting letters one by one or entering the complete word.

The player receives a limited number of attempts, making the game engaging and challenging. The application updates the interface dynamically and displays game results instantly.

---

## ✨ Features

✅ GUI-based interactive gameplay
✅ Random word generation
✅ Letter and full-word guessing
✅ Limited attempts (lives system)
✅ Real-time game updates
✅ Win/Loss result display

---

## 🧠 Algorithm

1. Start the application and display the game window.
2. Select a random word from a predefined list.
3. Show blank spaces for hidden letters.
4. Accept user input (letter or word).
5. Compare input with the selected word.
6. Reveal letters if correct.
7. Reduce lives if incorrect.
8. Update the game screen.
9. Check win or lose conditions.
10. Display final result and end the game.

---

## 🛠️ Concepts Used

* Python Lists
* Conditional Statements
* Loops
* Functions
* Random Module
* Tkinter GUI

# 📈 Task 2 – Stock Portfolio Tracker

## 📝 Description

The **Stock Portfolio Tracker** is a GUI-based investment tracking system that allows users to enter stock symbols and quantities to calculate total investment value.

The application uses predefined stock prices and automatically computes portfolio totals. Users can also save portfolio records into a text file for future reference.

This project demonstrates how Python can be used for financial calculations and data management.

---

## ✨ Features

✅ User-friendly GUI
✅ Stock and quantity input
✅ Automatic investment calculation
✅ Portfolio summary generation
✅ Save records into text files
✅ Data validation support

---

## 🧠 Algorithm

1. Launch the application.

2. Store stock prices in a dictionary.

3. Accept stock name and quantity.

4. Validate user input.

5. Verify stock availability.

6. Calculate investment value:

   **Investment = Stock Price × Quantity**

7. Add value to total portfolio.

8. Display portfolio details.

9. Save data into a file.

10. Show confirmation message.

---

## 🛠️ Concepts Used

* Dictionaries
* File Handling
* Arithmetic Operations
* Input Validation
* Tkinter GUI

# 🖼️ Task 3 – JPG File Automation Script

## 📝 Description

The **JPG File Automation Script** is an automation-based desktop application developed to organize image files efficiently.

Users select a folder, and the application automatically identifies all `.jpg` files and moves them into a separate destination folder. This reduces manual effort and improves file organization.

This project demonstrates practical automation using Python.

---

## ✨ Features

✅ Folder selection interface
✅ Automatic JPG detection
✅ File movement automation
✅ Destination folder creation
✅ Operation tracking
✅ User notifications

---

## 🧠 Algorithm
1. Start the application.
2. Allow the user to choose a folder.
3. Validate selected folder.
4. Create destination folder if needed.
5. Scan all files in the source directory.
6. Identify `.jpg` files.
7. Move image files automatically.
8. Count transferred files.
9. Display success results.
10. Finish execution.

---

## 🛠️ Concepts Used

* OS Module
* File Handling
* Automation
* Exception Handling
* Shutil Module
* Tkinter GUI

# 🤖 Task 4 – Basic Chatbot

Users enter messages, and the chatbot analyzes the text and responds using predefined rules. This project demonstrates the fundamentals of chatbot development and conversational logic.

It acts as an introduction to human-computer interaction systems.

---

## ✨ Features

✅ Interactive chat interface
✅ Rule-based response system
✅ Real-time communication
✅ User input handling
✅ Simple conversation flow

---

## 🧠 Algorithm 

1. Launch the chatbot interface.
2. Accept user message input.
3. Convert text into a standard format.
4. Compare message with predefined conditions.
5. Generate matching response.
6. Display chatbot reply.
7. Clear input field.
8. Continue until application closes.
